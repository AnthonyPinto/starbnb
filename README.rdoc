= Starbnb

Clone of the Airbnb website (circa 2014)

http://starbnb.herokuapp.com

== Primary Features

* Auth sytem with guest login

* interactive listing filtration with interactive map feed-back

* Live user creation with avatar selection and upload

* Live listing creation with image and location selection

== Run Locally

to run locally site requires figaro application.yml file with api keys for:

  GOOGLE_KEY

  FILEPICKER_KEY
